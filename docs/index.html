1.1 Goal 강의 목표

  • 이공계열 대학원생이 보다 쉽게 통계 이론을 습득하고 활용하는 능력을 배양하는데 주요 목적이 있음. 데이터 분석용 프로그래밍언어인 R을 기반으로 한 학습을 통하여 프로그래밍 기술과 함께 데이터를 분석하는 능력을 습득함으로써 이공계 연구에 있어서 필수인 통계적 사고의 기초를 다지는데 그 목적이 있음.

1.2 This course

이 강좌는 온라인 (강의자료, 동영상) 강의를 기본으로 함. 기초 통계 이론의 개념 중심 설명과 R을 이용한 실습 예시가 강의자료와 동영상 형식으로 강의사이트에 제공됨. 매 회 강의에는 해당 강의 내용과 관련된 과제가 주어지며 이메일을 통해 문제 풀이를 제출 받음. 상황에 따라 강의 자료 및 동영상 업데이트 일정이 조정될 수 있음.

1.3 Tips

  • 눈으로 이해하지 않고 스스로 실습 필수
  • 각 명령줄이 어떻게/왜 작동하는지 이해하기
  • 인터넷 검색을 통한 다른사람의 코드 이해/적용 필요
